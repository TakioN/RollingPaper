# Rolling Paper
학교 프로젝트 과목에서 **<<롤링페이퍼>>** 앱을 제작하였다.
## 📘프로젝트 소개
SNS가 아닌 유저들간의 소통 가능한 앱을 제작하고자 하였다.
## ⏰프로젝트 기간
- 2024.05.08 ~ 2024.06.21
## 프로젝트 팀
- **정재우** : 카카오톡 로그인 API 연결 및 로그인 페이지 구현
- **김민석** : 메인 화면 구현 및 git 관리, firebase 관리
- **남경식** : 시작 화면 구현 및 공유 API 연결
- **이정빈** : 꾸미기 화면 구현 및 firebase 관리
